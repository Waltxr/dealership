# README

## Install

### Clone the repository

```shell
git clone git@github.com:Waltxr/dealership.git
cd dealership
```

### Install dependencies

```shell
bundle install
```

## Serve

```shell
rails s
```

## Setup DB

```shell
rails db:setup
```
