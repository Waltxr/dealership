# README

## Install

### Clone the repository

```shell
git clone git@github.com:Waltxr/dealership.git
cd dealership
```

### Install dependencies

```shell
bundle install
```

### Setup DB

```shell
bundle exec rails db:setup
```

### Serve

```shell
bundle exec rails s
```
